# Telecom Customer Churn Prediction

## Project Overview

This project predicts whether a telecom customer is likely to churn (leave the company) or stay using Machine Learning.

## Dataset

* Dataset: Telco Customer Churn Dataset
* Total Records: 7043 Customers
* Target Variable: Churn (Yes/No)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Machine Learning Workflow

1. Data Loading
2. Data Cleaning
3. Handling Missing Values
4. Encoding Categorical Features
5. Train-Test Split
6. Logistic Regression Model
7. Model Evaluation

## Model Performance

* Accuracy: 78.75%
* Evaluation Metrics:

  * Confusion Matrix
  * Precision
  * Recall
  * F1-Score

## Results

The model predicts whether a customer is likely to churn or remain with the telecom company based on customer demographics, services subscribed, contract type, and billing information.
